GitHub个人主页的搭建

今天突发奇想想在github上弄一个自己的个人主页,记录一下过程,以供后面有需要使用

1、新建一个仓库

2、输入仓库名Repository name,例如我输入cjsxams.github.com,到时候就可以通过cjsxams.github.io访问我的主页,如果你输入的是其他的,例如joexams,那么到时候访问主页的地址为cjsxams.github.com/joexams.具体访问的地址后面会再说到.描述Description随便填,这个后面会显示在你主页,后面可以再修改,点击Create repository进入下一步

3、点击Settings进入

4、这里你可以重新设置仓库名称,点击进入主页设置

5、选择自己喜欢的模版,然后publish

6、建好后回到第四步你可以看到你个人主页的URL

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout gh-pages

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.

解决xampp在本地运行响应时间长的问题

之前一直都在用xampp做为本地的web集成服务器,但是响应时间一直都在2S以上,一直都没找到问题,今天看到有人用wamp也遇到这个问题了,说是把链接数据库那的localhost改为127.0.0.1,试了一下速度果然提升了,再上网查了一下说是在windows7下面连接localhost实际上是连接IP6下面的::1,所以还可以改本地的hosts文件,把localhost改到127.0.0.1,这种方法在有多个项目的情况下比去每个项目改配置文件方便
